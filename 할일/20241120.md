# 게시물 수정버튼 수정
- 게시물 수정 페이지로 로직 변경 (`post_edit.html`)
- 게시물 수정 완료 후 변경한 디테일 페이지로 이동 (수정 반영됐는지 확인하기 위해서)

# 댓글 추가하기
- 댓글 추가기능 만들기
- 댓글 작성 후 댓글을 작성한 디테일 페이지로 이동

# 댓글 삭제하기
- 댓글 삭제 후 댓글을 작성한 디테일 페이지로 이동
